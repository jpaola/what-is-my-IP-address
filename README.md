# what-is-my-IP-address

A program written in JS and HTML to display your IP address in a formatted table.

# API

Using `https://ifconfig.co` the response data format looks as follows:

```
{
  "ip": "",
  "ip_decimal": "",
  "country": "",
  "country_iso": "",
  "country_eu": "",
  "region_name": "",
  "region_code": "",
  "metro_code": "",
  "zip_code": "",
  "city": "",
  "latitude": "",
  "longitude": "",
  "time_zone": "",
  "asn": "",
  "asn_org": "",
  "user_agent": {
    "comment": ",
    "raw_value": ""
  }
}
```
